# Tecnova QA

Repositorio con la solucion de los desafios de automatizacion.

## Desafios incluidos

- `Desafio IMDB`: pruebas web automatizadas con Playwright.
- `Desafio pokeapi`: pruebas automatizadas de API con Cypress.

## Instrucciones de ejecucion

Cada desafio tiene su propio archivo `README.md` con las instrucciones necesarias para instalar dependencias y ejecutar las pruebas.

- Instrucciones IMDB: `Desafio IMDB/README.md`
- Instrucciones PokeAPI: `Desafio pokeapi/README.md`

## Apoyo con IA

Durante el desarrollo se utilizo una herramienta apoyada en IA como soporte para analizar requerimientos, revisar errores, ordenar los pasos de ejecucion y documentar la solucion.
